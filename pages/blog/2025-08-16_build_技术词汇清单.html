技术词汇清单

部署（Deploy）

将本地或测试环境的代码发布到线上平台，使其成为可访问的服务。

元数据（Meta）

“Meta”源自希腊语前缀，意味着“超越”或“关于……的”。

在 HTML 中，meta 标签用于描述页面的额外信息。

例如：<meta name="viewport" ...> → 描述网页在不同设备上如何显示。

POST 请求

POST 请求是将数据发送到服务器的方式之一。

它是 HTTP 协议中的一种方法（Method），通常用于创建或提交数据。

JSON

JSON（JavaScript 对象表示法）是一种常用的数据交换格式，广泛用于不同系统间传递数据。

通常，API 返回的数据就是 JSON 格式，它用来描述资源的详细信息。


{
"id": 1,
"name": "Coco",
"email": "coco@example.com",
"isAdmin": false,
"createdAt": "2025-08-15T10:00:00Z",
"skills": ["HTML", "CSS", "JavaScript"],
"profile": {
    "avatar": "https://example.com/avatar.png",
    "bio": "一名前端爱好者，喜欢用代码创造有趣的东西。"
}
}

指针（Pointer）

指针是用来存储变量内存地址的变量。

printf是C语言的格式化字符串，%d指输出带符号的十进制整数，\n是添加的换行符。

int main() {
    int a = 10;   // 一个普通变量
    int *p;       // 一个指针变量（用来存地址）

    p = &a;       // 把 a 的地址存到 p 里

    printf("%d\n", a);   // 直接输出 a 的值：10
    printf("%d\n", *p);  // 通过指针输出 a 的值：10

    return 0;
}
