技术词汇清单

deploy

把本地/测试环境的代码发布到线上可访问的环境（或平台）的过程。

meta

这个单词本身是希腊语前缀，意思是“超越、关于……的”。

放到 HTML 里时，meta tag 的意思就是“关于这个网页的额外信息”。

例子：<meta name="viewport" ...> → 关于网页在不同设备上显示方式的信息

POST 请求

POST 请求就是把数据打包发送给服务器的一种方式。

它是 HTTP 协议里的一种方法（Method），用来创建或提交数据。

JSON

JSON（JavaScript Object Notation）是一种数据格式，用来在不同系统之间传递信息。

最常见的就是 API 返回给前端的一份对象数据，用来描述某个资源的详细信息。

指针（pointer）

存储变量内存地址的变量。

int main() {
    int a = 10;   // 一个普通变量
    int *p;       // 一个指针变量（用来存地址）

    p = &a;       // 把 a 的地址存到 p 里

    printf("%d\n", a);   // 直接输出 a 的值：10
    printf("%d\n", *p);  // 通过指针输出 a 的值：10

    return 0;
}
